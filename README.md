## Getting started

Using yarn makes the installation much quicker.

```sh
npm i -G yarn
yarn add -g vuepress@next
yarn install
```

Then, launch the project:

```sh
npm run docs:dev
```

Or to build it:

```sh
npm run docs:build
```
